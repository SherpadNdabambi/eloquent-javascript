Recursive Table

Project description

The drawTable function recursively draws a table by taking in an array of objects or arrays as an argument and returning a string representating the data in the array.

An array is represented as a table and an object is represented as a row. The properties of an object are represented as columns.

For example, the code 

console.log(drawTable(MOUNTAINS));

will produce the output: 

name         height country
------------ ------ -------------
Kilimanjaro    5895 Tanzania
Everest        8848 Nepal
Mount Fuji     3776 Japan
Mont Blanc     4808 Italy/France
Vaalserberg     323 Netherlands
Denali         6168 United States
Popocatepetl   5465 Mexico

And 

let series = [
    {
        title: "Rick and Morty",
        seasons: [{
            number: 1,
            episodes: [
                {
                    number: 1,
                    title: "Pilot"
                },
                {
                    number: 2,
                    title: "Lawnmower Dog"
                },
                {
                    number: 3,
                    title: "Anatomy Park"
                }
            ]
        }]
    },
    {
        title: "See",
        seasons: [{
            number: 1,
            episodes: [
                {
                    number: 1,
                    title: "Godflame"
                },
                { number: 2 }
            ]
        }]
    }
];

console.log(drawTable(series));

will produce 

    title                seasons

               number       episodes

                        1    Godflame
                        2
