Recursive Table

The drawTable function recursively draws a table by taking in an array of objects or arrays as an argument and returning a string representating the data in the array.

An array is represented as a table and an object is represented as a row. The properties of an object are represented as columns.

For example, the code

let series = [
    {
        title: "Rick and Morty",
        seasons: [{
            number: 1,
            episodes: [
                {
                    number: 1,
                    title: "Pilot"
                },
                {
                    number: 2,
                    title: "Lawnmower Dog"
                },
                {
                    number: 3,
                    title: "Anatomy Park"
                }
            ];
        }]
    },
    {
        title: "See",
        seasons: [{
            number: 1,
            episodes: [
                {
                    number: 1,
                    title: "Godflame"
                },
                {
                    number: 2
                }
            ];
        }]
    }
];

console.log(drawTable(series);
will return the following output: 
+--------------------------------------------------+
|     title      |             seasons             |
|----------------+---------------------------------|
|                | number |        episodes        |
|                |--------+------------------------|
|                |        | number |     title     |
| Rick and Morty |        |--------+---------------|
|                |   1    |    1   | Pilot         |
|                |        |    2   | Lanwmower Dog |
|                |        |    3   | Anatomy Park  |
|----------------+--------+------------------------|
|                | number |        episodes        |
|                |--------+------------------------|
|                |        | number |     title     |
| See            |        |--------+---------------|
|                |   1    |    1   | Godflame      |
|                |        |    2   |               |
+--------------------------------------------------+
